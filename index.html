Projects

Recommendation System

Data was collected from an application which had more than 5000 user and data was pre-processed to remove any unwanted object

This system was devided into two parts, one for the new user and another for the old user where I used KNN, Matrix factorization, Item item and User user collaboration and applied A/B testing

Human Emotion Recognition

Dataset contains seven types human emotion such as sad, angry, neutral, confused, fear, disgust, pleasant

Data was processed using librosa library

Extracted MEL, MFCC features and applied RNN, LSTM, CNN for classifications

Able to obtain accuracy 97.88\%

IDC Image Classification

Dataset contains two types of Invasive Ductal Carcinoma(IDC histopathological images; IDC and Non IDC

Applied different CNN architecture and pre-trained models like VGG16, Resnet for transfer learning and got an accuracy which is 96.6%

Premature Baby Care Service
